These are upgrade packages for the 6MHz 68000 CPU boards that are required to run XENIX 3.

Thanks to dlightman for curating and providing these packages.



6mhz Long Board XENIX 3 Upgrade

The upgrade package for the first 68000 CPU board found in the Model 16, sometimes referred to the "long" board or the "dog-eared" board.  



6mhz Short Board XENIX 3 Upgrade

The upgrade package for the second generation 6MHz 68000 CPU board introduced in the Model 16B, referred to as the "short" board.
