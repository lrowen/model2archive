The -r option releases any memory	that is being held by FORCEHI.-q	causes FORCEHI to run silently with no messages being displayed	if everything works like it should.  Any errors will be reported.-f	causes FORCEHI to run silently with no messages being displayed	regardless of how successful the operation is.command...	Will be executed as though it was the only command on the	command line.  This command is executed after FORCEHI has	reserved or released low memory.  If FORCEHI is unable to reserve	or release memory (see -r option), it will not execute the command	that was specified.  If the -f option is used, the specified command	will always be executed regardless of whether FORCEHI is able to	reserve or release memory successfully.NotesFORCEHI will not cause a driver that can only run in low memory to be usablein high memory.The SYSGEN command should not be used while FORCEHI has memory reserved.Copyright InformationFORCEHI, program and documentation, Copyright 1990, by M.
