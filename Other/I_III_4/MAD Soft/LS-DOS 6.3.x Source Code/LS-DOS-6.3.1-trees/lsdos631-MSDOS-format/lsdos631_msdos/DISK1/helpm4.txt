patch
This utility alters the contents of a disk file. There are two methods of using it, the "command line" method:

The patch commands are:
X'hhhh'=aa bb ...   Appends a load block to the end of the file. The load
                    address is hhhh. The bytes to be loaded are aa, bb, etc

REMOVE    Reverses a D PATCH by using the contents of the "Find" string
